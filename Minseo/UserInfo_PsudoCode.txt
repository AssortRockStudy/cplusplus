## 회원정보 입출력 프로그램

1. 콘솔이 켜지면 회원 등록, 회원 전부 출력, 끝내기를 선택할 수 있음
2. 회원 등록은 이름, 성별(남1, 여2), 나이를 입력받고 추가로 입력 받을 것인지를 물어보고 추가입력과 메인화면 분기
3. 회원 전부 출력은 여태 입력받은 회원을 모두 출력하면 메인화면으로
4. 끝내기 버튼을 누르면 콘솔창 종료

### 조건
- 절대 문자열 비교 하지 말 것
- 우리가 강의에서 저장, 불러오기를 배우긴 했지만 이 프로젝트는 모두 코드에서 배운 걸 기반으로만 할 것이기 때문에 데이터 저장, 불러오기도 절대 구현하지 말것

---

모두의코드 에서 배운 내용을 기반으로 작성하라고 했으니까, 모두의코드 에 나왔던 내용만 사용하자.

전체적인 계획은 while문 안에 switch 문을 넣고, 등록과 출력은 함수로 구현해서 함수를 switch 분기 별로 나누고 끝내기 버튼을 누르면 break; 하면 되겠다

### 분기 1: 회원 등록
함수 종료시점에 회원을 추가로 입력 받을 것인지 선택할 것 같은데... 이 것도 while문에 if문으로 등록을 계속 반복할 것인지 아닌지(break;) 선택하면 좋겠다.
아직 동적 할당도 배우지 않았고, struct, class도 배우지 않았으니 이름 성별 나이 배열을 각각 100칸으로 만들기
배열마다 i번째 인덱스 참조하고 100개 넘어가면 예외처리 한 번 해주면 좋을 듯


### 분기 2: 회원 전부 출력
이름 배열의 i번 째 인덱스 출력하고
성별 배열의 i번 째 인덱스 출력하고(대신 성별은 1이면 남, 2이면 여 로 출력해야겠지?)
나이 배열의 i번 째 인덱스 출력하고
이걸 데이터 입력 개수만큼 for문 돌리면 되겠네요.


### 분기 3: 끝내기
break;넣으면 끝
